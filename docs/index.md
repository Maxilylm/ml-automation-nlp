# ml-automation-nlp

NLP and text analytics. Text preprocessing, embeddings, topic modeling, sentiment analysis, NER, and text classification. Requires spark-core installed.

## Project tracking

This plugin's work is tracked in Jira project [PM3](https://blend360.atlassian.net/jira/core/projects/PM3/board). Filter the board by label `plugin:nlp` for this plugin's slice.
